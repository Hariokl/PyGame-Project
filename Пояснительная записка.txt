	СОДЕРЖАНИЕ
	1. Описание
	2. Задачи приложения и игрока. Цель приложения
	3. Портрет целевой аудитории
 	4. Структура приложения
	1. Описание
		Игра - Base Defense. В игре нужно будет уничтожать врагов, чтобы те не дошли до базы. Если враг дойдёт до базы, то её хп будет снижено. При хп базы равно 0, игра завершается
	2.1. Задачи игры:
		1) Победить игрока
		2) Повышать количество врагов
		3) Повышать хп врагов
		4) Возможность показывать FPS
		5) Несколько уровней
		6) Разные виды врагов
		7) Динамичный геймплей
	2.2. Задачи игрока:
		1) Защищать базу от врагов
		2) Строить башни для уничтожения врагов
		3) Уничтожать врагов
	2.3. Цель приложения
		Главной целью игры будет закончить как можно больше волн.
	3. Портрет целевой аудитории
		Целевой аудиторией игры являются люди, чей любимый жанр игры - Tower Defense. Аудиторией может являться человек с любым возрастом, так как игра не содержит крови, нецензурных слов и так далее.
	4. Структура приложения
		Приложение будет приятно выглядеть, при этом иметь простой, интуитивно понятный интерфейс.
